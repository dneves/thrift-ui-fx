# Thrift UI - JavaFX

## running

- install [https://thrift.apache.org/](https://thrift.apache.org/)

- clone project: 
git clone git@github.com:dneves/thrift-ui-fx.git

- run it:
```
cd thrift-ui-fx
mvn clean package
java -jar target/jfx/app/thrift-ui-fx-jfx.jar
```
